# Demo project plans

Two alternative approaches for showcasing [canton-devkit](https://github.com/bitdynamics-ab/canton-devkit) LocalNet usage. Both are currently **parked** — not yet implemented.

| Plan | Focus | Best for |
|---|---|---|
| [Plan 1: IOU Daml demo](./plan-1-iou-daml-demo.md) | Custom Daml contract + JSON Ledger API + React UI | Daml / app developers |
| [Plan 2: Wallet SDK demo](./plan-2-wallet-sdk-demo.md) | Wallet SDK + Amulet transfers + devkit config bridge | Wallet / token integrators |

Plans can coexist in the same repo as sibling directories (`iou-demo/` and `wallet-demo/`) or be implemented on separate branches.
